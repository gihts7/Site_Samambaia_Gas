# Site Samambaia Gás

Projeto Integrador desenvolvido para gerenciamento e realização de pedidos de uma revenda de GLP.

## Tecnologias

* PHP
* HTML
* CSS
* MySQL
* Docker

## Funcionalidades

* Cadastro de usuários
* Login
* Catálogo de produtos
* Montagem de pedidos
* Área administrativa para gerenciamento de produtos
